# v2.7 Finalization Report

Date: 2026-07-01

## Scope

This report records the closure state for the architecture review, upstream comparison, accuracy fixes, frontend audit, test architecture cleanup, documentation alignment, and final repository cleanup.

## Completed Work

### Core detection and removal

- Fixed asset-key resolution for Gemini `96-20260520`.
- Replaced the corrupt `96-20260520` alpha asset and added a hashed real-sample regression.
- Kept Doubao rectangular resources dynamic through `widthxheight` keys.
- Added candidate trial-removal validation.
- Added standard-anchor preservation during final ranking.
- Added direct detector free-mode confidence floor.
- Preserved independent multi-anchor matches by using overlap-only NMS.
- Consolidated overlap/upsert/ranking behavior in `candidateGeometry.js`.
- Restricted global detector dimensions to alpha maps supplied by the active pipeline, preventing cross-profile template leakage.
- Reduced removal bias by using calibrated standard gain behavior.
- Hardened weak-alpha and texture cleanup paths.
- Fixed worker pool state release and timeout resilience.
- Added explicit detection traces and removal reports so detected and actually changed counts cannot diverge silently.
- Removed unsafe edge cleanup and unreachable halo retry behavior; halo severity is diagnostic only.

## Remaining Risks

- Browser-level UI behavior still needs Playwright coverage.
- Real-sample coverage should be expanded beyond synthetic fixtures.
- Diagnostic and stress layers are slow by design and should not be added to default developer feedback.
- A future profile must satisfy real-asset, catalog, entry-point and real-sample admission requirements before registration.
- Halo diagnostics need a reference-delta replacement before they may drive automatic retry or rejection.

## Maintenance Rule

Future detection changes must update code, tests, and docs together. In particular, profile support, asset dimensions, threshold ownership, and test group membership must remain dynamically aligned with source metadata.
