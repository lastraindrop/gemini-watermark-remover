# Gemini Watermark Remover - 用户指南 (v2.6.0)

本工具用于对 Gemini、Doubao、DALL·E 3 生成图片进行本地检测、分析与去水印。当前实现包含五阶段检测管线、NMS 假阳性过滤、亚像素精修、粗扫重定位、自适应三维评分、多遍移除与 Alpha 增益校准等增强。

## 0. 如何打开

- **推荐**: 打开项目根目录的 `index.html`（自动跳转到 `dist/index.html`）
- **或** 直接打开 `dist/index.html`
- **或** 运行 `npx serve dist` 然后访问 http://localhost:3000
- 不要直接打开 `public/index.html`（缺少构建产物 CSS/JS）

## 1. 使用入口

### Web

适合交互式单图与少量批量处理。

1. 直接把图片或文件夹拖到网页窗口，或使用上传入口。
2. 选择 `Gemini`、`Doubao` 或 `AUTO`。
3. **v2.3**: 在"高级设置"中选择**性能模式**（快速/均衡/全面）— 均衡默认即可满足大多数场景，全面模式提升裁剪或非常规图片的检出率。
4. 点击齿轮图标打开高级设置，调节检测灵敏度和梯度防御滑块（通常保留默认值）。
5. 处理完成后切换比较视图查看结果。
6. 单图模式下可使用"重新处理"按钮尝试不同的预设/设置而无需重置工作区。
7. 批量结果会以 ZIP 形式下载，避免浏览器并发下载遗漏。

### CLI

适合批量处理、自动化和脚本集成。

```bash
node src/cli.js -i ./input -o ./output
node src/cli.js -i input.png -o output.png --noiseReduction --no-deepScan
node src/cli.js -i input.png -o output.png --json
```

### Python

适合嵌入到 Python 工具链或 GUI。

```python
from python.remover import GeminiWatermarkRemover

remover = GeminiWatermarkRemover("./")
results = remover.remove_watermark(
    "./input_dir",
    "./output_dir",
    deep_scan=True,
    noise_reduction=False,
)
```

## 2. 关键参数

- `profileId`：`gemini`、`doubao`、`auto`
- `deepScan`：启用梯度滤波（推荐开启）。开启后引擎会同时检查亮度 NCC 和 Sobel 边缘梯度相关，对纯噪声假阳性有防御效果
- `noiseReduction`：对输出进行更强的噪声抑制（3x3 盒式模糊预处理）
- `autoDownload`：单图与批量处理后的自动下载

### 2.1 高级参数 (Advanced Settings)

在 Web 界面点击“齿轮”图标可展开高级配置面板，适用于自动检测失败的特殊情况：

- **检测灵敏度 (Threshold)**：默认 0.18。调低该值可识别更淡的水印，但会增加误报风险。
- **梯度防御力 (Penalty)**：默认 0.30。控制梯度滤波的惩罚强度。如果水印非常淡且背景复杂，可适当调高此值（放宽惩罚）。
- **手动选区模式 (Manual Area)**：
    1. 开启“手动选区模式”开关。
    2. 输入水印的精确坐标 (X, Y) 与尺寸 (Width, Height)。
    3. 引擎将绕过自动搜索，直接对指定区域进行数学恢复。

## 3. 检测原理

当前流程为五层检测管线：

1. **Catalog 精确匹配**：查找官方目录尺寸（10% 缩放容差），使用目录配置的 logo 大小与边距。
2. **Scaled Catalog 近似匹配**：若精确匹配失败，尝试从最近官方尺寸等比缩放（宽高比容差 5%，缩放比容差 8%）。
3. **Heuristic 启发式补充**：根据短边像素数联合判断 tier 级别（0.5k/1k/2k/4k），生成模板配置候选。
4. **Adaptive 自适应搜索**：粗到细多尺度搜索，使用三维评分（空间 NCC × 0.5 + 梯度 NCC × 0.3 + 方差 × 0.2）。
5. **Global Fallback 全局回退**：全图扫描，受锚点位置容差和置信度阈值保护。

检测结果经 `decisionPolicy` 分层判定为 `direct-match` / `needs-validation` / `insufficient` 三个级别。

### 高级设置 (v2.3)

点击控制栏的齿轮图标可展开高级设置面板：

- **性能模式**：快速/均衡/全面三档。均衡为默认，适合大多数场景；全面模式搜索范围更广（90%）、抖动更大（±8px）、精细调整更细（±8px），可改善裁剪或非常规图片的检出率，但处理时间约为快速的 4 倍。
- **检测灵敏度**：控制探针置信度阈值（0.05–0.80），默认 0.18。降低阈值可提升检出率但增加误报风险。
- **梯度防御**：控制梯度惩罚强度（0.10–0.90），默认 0.30。增加可减少纹理背景的假阳性，但可能降低弱水印检出。
- **手动选区**：当自动检测失败时，可手动拖拽选取水印区域并重新处理。
- **高级参数覆盖 (v2.6)**：在手动画布下方，点击"高级参数覆盖"展开：
  - **Alpha 增益**（0.5–3.0，默认 1.0）：直接设定水印移除强度。淡水印可提升到 1.5–2.0；过高会产生暗斑。
  - **位置搜索范围**（0–30px，默认 10）：在选定位置周围搜索精确水印中心的范围。水印偏移较大时增大此值。

## 4. 常见问题

### Q1. 为什么有些图检测不到？

通常是因为：

- 图片经过重新导出，尺寸不再是官方目录的精确值 — **v2.3**: 缩放匹配阈值已从 0.35 降至 0.25，大幅改善此情况
- 水印非常淡，且背景纹理过于复杂 — **v2.3**: 局部对比度阈值已从 0.015 降至 0.008，弱水印检测更灵敏
- 图片经过了剧烈的缩放、裁切或压缩 — 尝试切换到"全面"性能模式
- 水印在非标准位置（如裁剪后残留）— **v2.3**: 搜索范围已从 75% 扩展至 90%

当前版本已经增强了近似尺寸、梯度滤波、方差评分改进、**粗扫重定位**（±16px 处理水印偏移）、**NMS 假阳性过滤**（防止同一区域多次移除损坏图像）。极端样本仍可尝试手动模式的高级参数覆盖。

### Q2. 水印去除了但图像出现了色斑/暗斑？

这是"过度移除"的典型表现。v2.6 已通过以下机制改善：
- **NMS 假阳性过滤**：检测到多个重叠候选时只移除置信度最高的
- **Halo 检测**：多遍移除中检测边缘光晕并自动停止
- **亚像素精修**：微调 alpha 图位置减少边缘偏差
- 如仍有问题，在手动模式下**降低 Alpha 增益**至 0.7–0.9 重试

### Q3. 水印偏移几像素导致检测不到？

v2.6 显著改善了位置容错：
- **粗扫重定位**：锚点 NCC 低时自动 ±16px 粗扫
- **扩大抖动范围**：均衡模式从 ±6px 提升到 ±10px
- 如仍失败，在手动模式下**增大搜索范围**至 20–30px

### Q2. 什么是性能模式？如何选择？

性能模式（快速/均衡/全面）控制检测的深度与速度：

| 模式 | 搜索范围 | 深度扫描 | 抖动 | 自适应 | 适用场景 |
|------|---------|---------|------|--------|---------|
| 快速 | 60% | 关闭 | ±5px | 关闭 | 已知尺寸的干净图片 |
| 均衡 | 75% | 开启 | ±10px | 自动 | 大多数场景（默认） |
| 全面 | 90% | 开启 | ±14px | 自动 | 裁剪、缩放、复杂背景、水印偏移 |

### 前后对比 (v2.6)

每张处理完成的卡片右上角显示 **"对比"按钮**：
- 点击可在**原始图像**和**处理结果**之间切换
- 再次点击恢复处理结果视图
- 帮助快速验证水印是否正确移除，检测微小偏差

选择模式后，`Deep Scan` 和 `Noise Reduction` 开关会自动同步到预设值。预设参数详情会实时显示在下方提示栏中。

### Q3. 单图处理完成后如何重新处理？

点击紫色的"重新处理"按钮，无需重置工作区。适合在切换性能模式或调整阈值后快速重试。

## 6. 维护原则

- 新增 profile 时，必须同步 catalog、detector、engine、测试与文档。
- 调整阈值时，必须同时检查误报和漏报。
- 梯度滤波的三个应用点（Phase 1 / Phase 2 / 抖动分支）必须同步修改。
- 调整前端参数时，必须同步 Web/CLI/Python。
- 版本号、测试总数、URL、包名等元信息在所有文档中必须一致。
- 如果某个数字是历史快照，必须明确标注为历史，不能写成当前基线。
